Trace inspector is waiting for daemon data

This page turns local daemon state into a human-readable view without changing how MetaBots talk over MetaWeb.

Snapshot

One place for the current A2A identifiers and public state.

Participants

Keep the subject as MetaBot-to-MetaBot, not endpoint-to-endpoint.

    Artifacts

    The browser is observational. The exported markdown and JSON remain the durable evidence.

    First-Class States

    These cases should be visible, not flattened away.

    Timeout The host session stopped waiting, but the remote MetaBot may still be working and can still complete later.
    Clarification The remote MetaBot requested one follow-up question inside the same A2A session.
    Manual Action A guarded branch needs a human decision instead of pretending the task can continue automatically.

    Transcript Timeline

    If session-level transcript records exist, show them in order. Otherwise fall back to exported markdown.

    Public Status Events

    These events come from the local daemon watch layer, not a browser-only animation.
